TYPES OF CODING LANGUAGES -

1) DECLARATIVE CODING LANGUAGE -

 The coding languages in which we can only use specific commands to create apps, games, websites and more, are known as Declarative Coding Languages. EG- C, C++, Java, Java Script.

2) IMPERATIVE CODING LANGUAGE-

The coding languages in which we can create and use our own commands to create apps, games, websites and more, are known as Imperative Coding Languages. EG- Html, Jsx, React.Js.
